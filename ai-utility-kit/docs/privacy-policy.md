# Privacy Policy

`ai-utility-kit` is a skill-only plugin. It does not include a background service, remote API client, telemetry collector, database, or file writer.

The plugin processes information only through the active Claude Code or Codex conversation. Users should avoid pasting confidential meeting records, personal data, customer data, credentials, contracts, or regulated information unless their runtime environment and organization policy allow that use.

The plugin may ask for additional context such as goals, audience, decisions, attendees, terms, risks, or source text. Those details are used only to produce the current answer inside the host application.

In v0.1, the plugin does not create files, publish content, call external services, or sync information to third-party systems.

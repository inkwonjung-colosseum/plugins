# meeting-brief

회의나 긴 대화 기록을 실행 가능한 요약으로 정리하는 스킬입니다.

## 언어

- 기본 출력은 한국어입니다.
- 사용자가 영어로 요청하면 영어로 답합니다.

## 호출

- Claude Code: `/ai-utility-kit:meeting-brief <회의 기록>`
- Codex: `$meeting-brief <회의 기록>`

## 입력 확인

가능하면 아래 정보를 확인합니다. 없으면 `미정`으로 표시하고 임의로 만들지 않습니다.

- 회의명
- 날짜
- 참석자
- 원문 또는 메모
- 원하는 출력 길이

## 동작 순서

1. 원문에서 주제, 결정, 요청, 우려, 후속 작업을 분리합니다.
2. 반복 발언, filler, 잡담은 줄이되 의미 있는 뉘앙스는 보존합니다.
3. 결정사항과 단순 의견을 구분합니다.
4. 액션아이템은 담당자, 할 일, 기한으로 나눕니다.
5. 담당자나 기한이 불명확하면 `미정`으로 표시합니다.

## 출력 형식

```md
## 요약
- <핵심 요약>

## 결정사항
- <결정된 내용>

## 리스크 / 오픈질문
- <리스크 또는 아직 답이 없는 질문>

## 액션아이템
| 담당자 | 할 일 | 기한 |
|---|---|---|
| 미정 | <할 일> | 미정 |
```

## 제한

- 이 스킬은 파일을 생성하거나 수정하지 않습니다.
- 원문에 없는 참석자, 결정사항, 담당자, 기한을 만들어내지 않습니다.
- 민감한 인사, 법무, 보안, 고객 데이터가 있으면 필요한 최소 정보만 요약하도록 안내합니다.

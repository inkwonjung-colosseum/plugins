# term-clarifier

문서나 대화에서 섞여 있는 용어를 정리하고, 같은 의미의 다른 표현과 애매한 표현을 분리하는 스킬입니다.

## 언어

- 기본 출력은 한국어입니다.
- 사용자가 영어로 요청하면 영어로 답합니다.

## 호출

- Claude Code: `/ai-utility-kit:term-clarifier <정리할 내용>`
- Codex: `$term-clarifier <정리할 내용>`

## 동작 순서

1. 핵심 명사, 행위, 상태, 역할, 산출물을 뽑습니다.
2. 같은 의미로 보이는 표현과 서로 다른 의미인데 같은 단어를 쓰는 표현을 찾습니다.
3. 추천 표준 용어를 제안합니다.
4. 확정된 정의와 아직 확인이 필요한 정의를 구분합니다.
5. 개념 간 관계를 짧게 정리합니다.

## 출력 형식

```md
## 추천 용어
| 추천 용어 | 정의 | 피할 표현 |
|---|---|---|
| <용어> | <한 문장 정의> | <동의어 또는 모호한 표현> |

## 애매한 표현
- <표현>: <왜 애매한지와 확인 질문>

## 개념 관계
- <용어 A>는 <용어 B>와 <관계>입니다.

## 확인 필요
- <사용자나 도메인 담당자에게 확인할 항목>
```

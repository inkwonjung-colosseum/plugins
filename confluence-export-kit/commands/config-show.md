# Config Show

Display the current `confluence-markdown-exporter` configuration with `cme config list`.

## Usage

```text
/confluence-export-kit:config-show [--json]
```

- `--json` — output config as JSON instead of the default human-readable format

See `skills/config-show/SKILL.md` for runtime rules and response format.

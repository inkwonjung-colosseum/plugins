# Terms of Service

`confluence-export-kit` provides local Confluence export and export-index workflows for Claude Code and Codex.

The plugin is a convenience wrapper around local configuration, Confluence export, and local Markdown indexing. Users remain responsible for confirming that they have permission to export the selected Confluence content and store it in the chosen workspace.

The plugin does not publish documents back to Confluence, make Jira changes, grant access to restricted pages, or guarantee that local exports are current after the source content changes.

By using the plugin, users agree to verify exported content, credential handling, and local file retention against their own organization policies before relying on or sharing generated exports.

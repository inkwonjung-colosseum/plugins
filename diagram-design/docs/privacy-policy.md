# Privacy Policy

`diagram-design` is a local workflow plugin for creating technical and product diagrams as standalone HTML files with inline SVG.

The plugin processes information only through the active Claude Code or Codex conversation and local workspace. It may use user-provided descriptions, system details, style preferences, and optional reference URLs or project files to guide diagram design.

In v1.0, the plugin does not include a background service, telemetry collector, hosted database, or automatic publishing pipeline. Generated diagram files remain in the user's local workspace unless the user separately shares or publishes them.

Users should avoid including confidential architecture details, credentials, customer data, or regulated information in diagrams unless their runtime environment and organization policy allow that use.

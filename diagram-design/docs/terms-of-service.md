# Terms of Service

`diagram-design` provides guidance for creating technical and product diagrams as standalone HTML and inline SVG artifacts.

The output is advisory and visual. Users remain responsible for verifying architectural accuracy, product meaning, branding rights, accessibility, and whether a generated diagram is appropriate to publish.

The plugin does not replace architecture review, security review, legal review, brand approval, or documentation approval. It also does not automatically publish diagrams to external systems in v1.0.

By using the plugin, users agree to review generated diagrams and any included system details before sharing them with internal or external audiences.

# Privacy Policy

`logistics-expert-kit` is a skill-only plugin. It does not include a background service, remote API client, telemetry collector, or file writer.

The plugin processes information only through the active Claude Code or Codex conversation. Users should avoid pasting confidential contracts, freight rates, customer data, carrier credentials, or regulated shipment details unless their runtime environment and organization policy allow that use.

The plugin may ask for additional context such as process steps, order volume, lead time, inventory counts, carrier mode, and exception symptoms. Those details are used only to produce the current answer inside the host application.

This plugin does not guarantee legal, customs, tax, safety, contract, or regulatory conclusions. When those topics appear, it directs users to responsible internal owners or qualified external specialists.
